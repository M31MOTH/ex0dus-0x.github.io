D0xk1t

This is a tool I've created in python for the sole purpose of writing a D0x. D0xing is viewed as script-kiddish, illegal and harmful. People often look down at those who write D0x. However, this tool is created not to harm. PyPayload

This is a tool designed specifically for the task of automation. It allows the ease of creating payloads using Metasploit's msfvenom, which combines the task of payload creation and encoding. Users specify the flags, and the program does the rest. Many payload options are available, but only currently for Windows. Future support for Android and Linux.

dedsploit

Won Privacy Hack at defhacks() 2016

In collaboration with JonW27, we built a framework that targets network protocols and services. It utilizes a Metasploit interpreter-like shell, and several modules that represent different network protocols. Looking at the tree view of all the modules and its attack vectors: 
  main
|
+--SSH
|    +--
|       |- bruteforce - bruteforce vulnerable SSH server
|
+--SMTP
|     +--
|        | - bruteforce - bruteforce SMTP address (aka email)
|        |
|        | - smsbomb - utilizes smtp-to-email gateway to spam SMS messages
|
+--HTTP
|      +--
|         | - arpspoof - MITM where user fakes ARP messages on LAN, intercepting packets from host
|         |
|         | - slowloris - Layer-7 DoS attack using slow headers and malformed GET requests to a vulnerable web server
|
+--Recon
|       +--
|          | - pscan - port scan with Nmap
|          |
|          | - hosts - scan for active hosts
|
+------
This project will be continously worked on, as many of the modules are incomplete, and there are always going to be minor syntatical errors. Official Site Here!
