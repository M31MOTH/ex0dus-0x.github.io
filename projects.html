PyPayload v1 experimental

This is a tool designed specifically for the task of automation. It allows the ease of creating payloads using Metasploit's msfvenom, which combines the task of payload creation and encoding. Users specify the flags, and the program does the rest. Many payload options are available, but only currently for Windows. Future support for Android and Linux.
